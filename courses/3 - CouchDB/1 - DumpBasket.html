DumpBasket

Before starting:

  • Install Apache CouchDB
  • Make sure you have your preffered HTTP tool (Postman, cURL...)
  • Do the CouchDB tutorial in the manuals (available through Futon's manuals)

After having a basic understanding of what Apache CouchDB is and how to communicate with it, you will use it to create an online file share solution named DumpBasket.

All the data in our system is shared between everyone, permissions are not supported, we want to share everything!

The entire data will be saved inside a single CouchDB database and you will create a simple UI using a framework of your choice to allow directory browsing of the data.

DumpBasket will support two types of items - files and folders.

File

Our system only supports text files, files have a name and they must live in a folder.

Folder

A folder is a container for files, it also has a name.
