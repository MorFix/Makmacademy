Game World

This exercise will combine everything you have learned so far in larger scale.

You will create an In-Browser game using HTML, CSS & JS.

Your job is to implement the best Minesweeper ever created!

Required features

left mouse button
  • click: open the tile if not flagged.
  • double click: reveal all surrounding tiles if all flags are placed (with the possibility to loose).
right mouse button
  • click: switch between flag, '?' and empty tile.

Notes

  • look for the game's rules in the internet
  • don't forget to implemenet a settings menu
  • make it pretty! (did someone mention animations?)
  • lint your code
  • do not use canvas
