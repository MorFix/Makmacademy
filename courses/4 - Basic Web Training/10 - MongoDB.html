MongoDB Querying

In this part of the exercise you will write some MongoDB queries.

Download the social.js script to your computer and run it using the mongo shell.

Obviously, you have to install MongoDB on your machine.

The script will create a new database named 'tar' with a single collection named users.

Write queries that will return the following data:

  1. All users.
  2. A single female user.
  3. All users sorted by birthdate descending.
  4. All users, showing only the name field.
  5. The 2 oldest users.
  6. All users who posted a status which received more than 5 likes.

Write queries for the following data using mongo's aggregation framework:

  1. All users.
  2. Number of female users and male users (Two values, same query).
  3. Total number of likes in the entire database.
