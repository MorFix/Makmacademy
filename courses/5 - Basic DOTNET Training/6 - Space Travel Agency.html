Space Travel Agency

This exercise should be done using Windows Forms

In this exercise you will create an application for a space travel agency, the agency allows users to fly in a spaceship.

The supplied DLL (Download button above) does all the heavy-lifting for actually managing the spaceships, you will use it to order and launch spaceships for the user, but launching spaceships is hard work and we want our client to be responsive so you will make sure the heavy-lifting is done in parallel and the main thread is free to respond to the user.

Ordering and launching a spaceship consists of the following stages in this order:

  1. Ordering a spaceship
  2. Unlocking the spaceship using a password and a fingerprint together
  3. Ordering a pilot
  4. Assembling a crew of 5 aliens, the entire crew must be assembled together otherwise no one will come
  5. Launching the spaceship

When needed run methods on the threadpool, and when needed use tasks.
