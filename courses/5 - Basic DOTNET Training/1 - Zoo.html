The Zoo

Welcome to our amazing .NET training!

You are going to create a migratable zoo with many fun animals.

Our zoo will have the following animals:

  1. Elephant - a mammal with the following properties:
    • TrunkLength
    • Tusks - a number represeting the number of tusks
  2. Tiger - a mammal with the following properties:
    • Stripes - number of stripes on the tiger
    • HumansEaten - number of humans eaten
  3. Otter - a mammal with the following properties:
    • FavoriteRock - another object representing the otter's favorite rock
      • Unfortunately the Favorite rock has weight
  4. Shark - a fish with the following properties:
    • Type - the shark's type, can be one of the following:
      1. GreatWhite
      2. Hammerhead
      3. Loan
    • IsLawyer
  5. Chameleon - a lizzard with the following properties:
    • CurrentColor
  6. Ostrich - a big-ass bird with the following properties:
    • IsHeadInGround

In addition to this, all animals have the following properties:

  • Name
  • Age
  • Gender
  • Favorite human starting with the first letter of their name

Pay attention to correct inheritance in your classes.

After creating the classes, we need our zoo to be migratable. In order to do that our animals must be serializable.

To do that, we will create an interface called ISerializableObject with a single method returning a dictionary with the animal's properties and values. (DO NOT USE ANY FRAMEWORK INTERFACE/CLASS!!1) For example, the basic implementation for Animal will return: { "Name": "Rexi", "Age": 5, "Gender": "Male" }

After implementing the ISerializableObject in your classes, we will create the following serializers:

  • CsvSerializer - Serializes the objects to a CSV file (one csv file for all the animals in the zoo)
  • JsonSerializer - Serializes the objects as JSON to a file (one json for all the animals in the zoo) Download the examples to check if your serializers work correctly

NOTE: Don't use reflaction. NOTE: When using the file system, some errors may occur, make sure you handle them correctly.
